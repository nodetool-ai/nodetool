# NodeTool v0.6.3-rc.11 Changelog

## User Interface

### Workflow Editor Enhancements
- **Improved Node Duplication**: Duplicate nodes now preserve upstream connections, maintaining workflow connectivity when copying node groups. Also should not mess with edges anymore that don't belong to them.
- **Enhanced Selection Handling**: Improved selection management for Group nodes. Improved Group selected style.
- **Find in Workflow Dialog**: Added close button and close on click outside. 

### Text Editor & Clipboard
- **Escape Key Support**: Escape closes the Text Editor Modal again.
- **Markdown Paste Handling**: Fixes pasted text overwriting existing text.

### Asset Management
- **Asset Update Handling**: Improved asset update flow now preserves previous values and includes automatic refetching after changes. Improves renaming and moving files.

### Realtime Audio
- **Fixed Realtime Audio UI**: Resolved display issues with the realtime audio interface.

## Nodes & Integrations

### New FAL Nodes
- **Pixverse v5.6**: New text-to-video, image-to-video, and video transition nodes.
- **Hunyuan Image v3 Instruct**: Advanced image generation with instruction-following capabilities.
- **LTX-2 19B Audio-to-Video**: Generate videos synchronized with audio input.
- **Bria Replace Background**: Image background replacement.
- **Qwen 3 TTS**: Text-to-speech synthesis.
- **Qwen Image Max Edit**: Image editing.
- **Video Face Swap**: Swap faces in video content.
- **Z-Image Nodes**: Image restoration and enhancement.
- **Face Swap Nodes**: Additional face manipulation capabilities.

### Node Updates
- **KIE Flux2**: Now supports multiple input images (1-8 images), removed unsupported parameters: steps and guidance_scale.
- **GPT Image 1.5**: Both TextToImage and ImageToImage nodes now support multiple input images with updated size options.

## Core & Backend Improvements

### Stability & Performance
- **WebSocket Concurrency Fix**: Implemented send lock in the unified WebSocket runner to prevent frame interleaving during high-frequency messaging.
- **SQLite Connection Improvements**: Enhanced connection pool management with write lock serialization and timeout features to prevent deadlocks.

### Provider Updates
- **Mistral Provider**: Added provider registration with regression tests.
- **Gemini Model Names**: Fixed model names to match Google GenAI SDK validation requirements.

### Technical
- **Type Safety**: Fixed 20+ type checker warnings across providers, models, and tools.
- **Provider Initialization**: Improved provider initialization and removed deprecated features.
- **Agent Output Handling**: Refined to yield text only when content is present, preventing empty outputs.
- **Vite Build Optimization**: Improved dependency optimization and build configuration.
