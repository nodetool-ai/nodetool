# NodeTool

NodeTool is an open-source visual workspace for building and running AI workflows. It connects image, video, audio, language, agent, and data models on a node-based canvas.

## Editions

- [NodeTool Studio](https://nodetool.ai/studio.md) is the free desktop application for macOS, Windows, and Linux.
- [NodeTool Cloud](https://nodetool.ai/cloud.md) is the hosted browser application.
