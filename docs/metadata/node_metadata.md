# nodetool.metadata.node_metadata

## NodeMetadata

Metadata for a node.

- **title** (str)
- **description** (str)
- **namespace** (str)
- **node_type** (str)
- **primary_field** (str | None)
- **secondary_field** (str | None)
- **layout** (str)
- **properties** (list)
- **outputs** (list)
- **model_info** (dict)
