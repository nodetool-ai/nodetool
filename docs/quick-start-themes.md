## Tips

### Try Different Themes for Different Tasks
- **Coding for long hours?** Try Ocean or Forest
- **Creative design work?** Try Sunset or Midnight
- **Working in bright light?** Try Light mode
- **Late night session?** Try Dark or Midnight

### Theme Affects Everything
Your theme changes colors throughout NodeTool:
- Nodes and edges in the workflow editor
- Panels and menus
- Text and backgrounds
- Buttons and controls

### Switching is Easy
Change your theme anytime - it only takes 2 clicks!

## Which Theme Should I Choose?
