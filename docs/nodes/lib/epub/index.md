This namespace contains 4 node(s).

## Available Nodes

- **[EPUB Extract Chapters](epubextractchapters.md)** - Extract each chapter as a separate text item with its title.
- **[EPUB Extract Text](epubextracttext.md)** - Extract plain text from an EPUB file by concatenating all chapters.
- **[EPUB Metadata](epubmetadata.md)** - Read metadata (title, author, language, publisher, etc.) from an EPUB file.
- **[EPUB Table of Contents](epubtableofcontents.md)** - Get the table of contents (chapter titles, ids, hrefs) from an EPUB file.
