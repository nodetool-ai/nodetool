**Type:** `lib.hashlib.HashString`

## Description

Compute the cryptographic hash of a string using hashlib.
    hash, hashlib, digest, string

    Use cases:
    - Generate deterministic identifiers
    - Verify data integrity
    - Create fingerprints for caching
