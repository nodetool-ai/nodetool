This namespace contains 5 node(s).

## Available Nodes

- **[Add / Subtract Time](addsubtracttime.md)** - Add (or subtract, when amount is negative) a number of time units to a date.
- **[Date Difference](datedifference.md)** - Difference between two dates (date_a − date_b) expressed in the given unit.
- **[Format Date](formatdate.md)** - Parse a date string/number/Date and format it. Supports tokens YYYY, MM, DD, ...
- **[Now](now.md)** - Return the current date as ISO string, epoch ms, and a Date value.
- **[Start / End of Period](startendofperiod.md)** - Return the start and end of the given period (day, week, month, year).
