**Type:** `lib.svg.Transform`

**Namespace:** `lib.svg`

## Description

Apply transformations to SVG elements.
    svg, transform, animation

    Use cases:
    - Rotate, scale, or translate elements
    - Create complex transformations
    - Prepare elements for animation

## Properties

| Property | Type | Description | Default |
|----------|------|-------------|----------|
| content | `any` | SVG element to transform | - |
| translate_x | `any` | X translation | `0` |
| translate_y | `any` | Y translation | `0` |
| rotate | `any` | Rotation angle in degrees | `0` |
| scale_x | `any` | X scale factor | `1` |
| scale_y | `any` | Y scale factor | `1` |
