**Type:** `lib.markitdown.ConvertToMarkdown`

**Namespace:** `lib.markitdown`

## Description

Converts various document formats to markdown using MarkItDown.
    markdown, convert, document

    Use cases:
    - Convert Word documents to markdown
    - Convert Excel files to markdown tables
    - Convert PowerPoint to markdown content

## Properties

| Property | Type | Description | Default |
|----------|------|-------------|----------|
| document | `any` | The document to convert to markdown | `{'type': 'document', 'uri': '', 'asset_id': None, 'data': None}` |
