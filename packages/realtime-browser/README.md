**Reserved for Phase 3 (WebRTC client transport).** This package is not imported by `web/` or `packages/websocket/` in the Phase R MVP.

Phase R uses frame_push mode over the existing WebSocket. When Phase 3 ships a WebRTC client transport, the browser-side WebRTC/codec logic will live here.

Do not add imports from this package to `web/` or `packages/websocket/` until Phase 3 begins.
